Everything you need, nothing you don't

A complete deployment platform that lives entirely in GitHub. No dashboards to maintain, no APIs to learn.

🌐

GitHub is the Control Plane

Push to deploy. Open a PR to get a preview environment. Close the PR to clean it up. GitHub Actions handles everything.

📦

Multi-App Hosting

Run multiple applications on a single server. Each gets its own subdomain, database, storage bucket, and isolated containers.

🔒

Auto TLS

Caddy handles HTTPS automatically. Every app and preview environment gets a valid TLS certificate with zero configuration.

🔍

Built-in Observability

Centralized logging with Loki and Promtail. Grafana dashboards out of the box. Automated health alerts via GitHub Issues.

🔀

Fork to Self-Host

Fork any app repository, configure 4 secrets, push. You're running your own instance. That's the entire self-hosting guide.

🛡️

Production Ready

Daily backups, automated security updates, resource limits, credential isolation, and firewall configuration out of the box.

Three steps to production

Go from zero to deployed in minutes, not days.

1

Bootstrap

Run a single script on a fresh Debian server. It installs Docker, PostgreSQL, Redis, Caddy, and the observability stack.

2

Create

Generate a new app from the template. It comes with FastAPI, database migrations, Docker config, and CI/CD workflows pre-configured.

3

Push

Set 4 GitHub secrets and push to main. GitHub Actions builds, deploys, and configures your app automatically.
